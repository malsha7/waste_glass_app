# Waste Glass Collection System

##  Project Title
WasteGlassAPI – Glass Collection & Route Optimization System


## 📖 Project Description
Waste Glass Collection System is a mobile application integrated with a backend REST API to manage glass recycling operations efficiently.  

The system supports supplier validation, collection entry, optimized route planning, and trip report analytics. The backend is hosted on **Render cloud platform**, providing live API access.

##  Features

- Supplier validation using supplier code
- Glass collection entry (clear & coloured glass)
- Route optimization for collection vehicles
- Trip report analytics dashboard
- Clean and modern UI design
- Real-time API integration with backend
